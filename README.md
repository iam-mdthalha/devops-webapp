# DevOps CI/CD Pipeline Project

## Description
Automated CI/CD pipeline using Jenkins, Docker, and AWS EC2

## Technologies
- Node.js + Express
- Docker
- Jenkins
- AWS EC2
- Nginx
- GitHub Webhooks

## Architecture
Developer → Git Push → GitHub Webhook → Jenkins → Docker Build → DockerHub → Production Server
